# Hospital Management System - Frontend

This is the React frontend for the Hospital Management System, built using Vite.

## Features
- Patient Dashboard
- Doctor Appointments
- Specialty Selection
- Responsive UI

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment
This project is configured to be deployed on **Vercel**.
- Connect this repository to your Vercel account.
- Set the build command to `npm run build`.
- Set the output directory to `dist`.
